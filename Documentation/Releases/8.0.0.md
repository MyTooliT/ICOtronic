# API

- Removed code to convert data into multiples of g. For more information on why, please take a look at [this issue comment in ICOapi](https://github.com/MyTooliT/ICOapi/issues/29#issuecomment-4668940786).
- Remove code for reading and writing EEPROM calibration data (constants for conversion into g). The calibration data was never written at production, as far as I can tell. The latest version of the [hardware test suite](https://github.com/MyTooliT/ICOtest) also does not write calibration data into the EEPROM.
